Paddle speeds will vary depending on the machine

If you want to change the paddles and the ball, look in the Data/Extras for provided alternatives 
and replace the files in the Data/Objects directory. If you want to use video memory instead of system memory, 
edit the Data/config file

Source code is in the src directory, for help with installing and linking SDL, go to: 
 http://lazyfoo.net/SDL_tutorials/lesson01/index.php
